# Kolla – Frontend (Vue 3)

## Zweck
Dieses Repository enthält das Frontend des Prototyps „Kolla“, einem kollaborativen Aufgabenmanagementsystem im Rahmen des Moduls „Softwarearchitekturen und Qualitätssicherung (SAQS)“ (TH Brandenburg).   
Die Implementierung ist prototypisch und dient dazu, die entworfene Architektur sowie das Zusammenspiel von GUI und Anwendungslogik zu demonstrieren; es handelt sich nicht um ein produktives System. 

## Projektkontext (SAQS)
Kolla soll als Web‑Applikation Akteure bei der Aufgabenbearbeitung (inkl. Priorisierung/Workflow) unterstützen und dabei eine flexibel austauschbare GUI ermöglichen.   
Der Prototyp fokussiert die Machbarkeit und die saubere Trennung zwischen GUI und Anwendungslogik. 

## Start (lokal)
- Abhängigkeiten installieren: `npm install` 
- Dev‑Server starten: `npm run dev`
